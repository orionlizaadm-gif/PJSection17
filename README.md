# PJSection17 Room Rental Landing Page

A simple HTML landing page for promoting furnished room rentals in Section 17, Petaling Jaya.

The primary goal of this website is to convert visitors into WhatsApp enquiries.

---

## Features

* Mobile responsive
* Minimalist black design
* WhatsApp call-to-action buttons
* Floating WhatsApp button
* Room gallery section
* FAQ section
* SEO-friendly structure
* Fast loading (single HTML page)

---

## Recommended Image Guidelines

For best conversion results:

### Hero Image

Use the brightest and cleanest room photo.

Include:

* Bed
* Window
* Air-conditioning
* Clean walls

Avoid:

* Bathrooms
* Kitchen-only shots
* Corridor photos

---

### Bathroom Photos

Private attached bathrooms are a major selling point.

Use:

* Clean
* Bright
* Wide-angle photos

---

### Shared Facilities

Recommended photos:

* Kitchen
* Dining Area
* Garden
* Pond / Water Feature

These help visitors imagine living comfortably in the property.
